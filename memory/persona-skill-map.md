## 🚀 加载规则

### 触发条件
1. 用户明确提到任务类型（习惯/健康/学术/技术/创意等）
2. 用户要求"以XX角色"或"像XX一样"
3. 用户提到特定领域关键词

### 加载流程
1. 根据任务类型从映射表选择对应技能
2. 使用 `skillhub_install` 安装技能
3. 读取技能 SKILL.md 加载人格设定
4. 在当前会话中应用该人格

### 当前已安装（可优先使用）
- 内置雨凌音人格（习惯/情感陪伴）
- agent-mbti（人格诊断）
- academic-writing
